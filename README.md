# 📊sales-analytics-dashboard
Sales Analytics Dashboard with Power BI

## Description
Dashboard Power BI d'analyse des ventes 
couvrant la période Mars 2024 - Mars 2025.

## Dataset
- 897 transactions
- 5 catégories de produits
- 3 modes de paiement

## KPIs
- Total CA : $665,65K
- Moyenne Score : 3,98
- Total Quantités : 2640
- Total Commandes : 897
- Panier Moyen : $742,08
- Nombre Clients : 897

## Modèle de données
Modèle en étoile avec :
- fait_vente (table de faits)
- dim_date, dim_client, dim_produit
- dim_categorie, dim_ville, dim_paiement

## Outils utilisés
- Power BI Desktop
- DAX
- Power Query
